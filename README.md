# Linux Automation

A growing collection of Linux automation scripts, Bash utilities, and command-line tools built for learning, experimentation, and real-world system tasks.

This repository serves as my personal automation lab where I practice scripting, Linux administration, monitoring, networking, and workflow automation across multiple environments.

---

## 📌 About

This project contains automation scripts and utilities written mainly in:

- Bash
- Shell scripting
- Linux command-line tools

The repository is continuously updated with new scripts and experiments covering different areas of Linux and system automation.

---

## 💻 Supported Environments

Scripts are tested across different platforms including:

- Termux (Android)
- Red Hat Enterprise Linux (RHEL)
- Kali Linux
- Ubuntu
- Windows (via WSL/Git Bash where applicable)

Because of environment differences, some scripts may require slight modifications depending on the operating system or installed packages.

⚙️ Getting Started

Clone the repository

git clone git@github.com:Abeltrust/linux-automation.git

Move into the project directory

cd linux-automation

Make scripts executable

chmod +x *.sh

Run a script

./script-name.sh


---

🛠 Requirements

Depending on the script, you may need:

Bash shell

Linux utilities (curl, grep, awk, sed, etc.)

Git

Internet connection (for networking scripts)

Root/sudo privileges for some system-level tasks
